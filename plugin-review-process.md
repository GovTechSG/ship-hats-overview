## Minimum requirement for a plugin request

- Data center approved
- *Cloud Supported
- Atlassian endorsed as Trusted Partners

  - *Cloud Security Participant
- Plugin Ratings:
  - Stars > 3
  - Votes > 50
- Vendor Supported

*Not applicable to Bamboo
